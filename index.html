Como usar

Estatística descritiva

Conceito: A estatística descritiva é um ramo da estatística que aplica várias técnicas para descrever e sumarizar um conjunto de dados.

Título da pesquisa: O título da pesquisa serve para dar nome ao que está sendo pesquisado.

Valores a ser pesquisado: Os valores podem variar com o que deseja pesquisar como idades (Números: 16;48;18;68) ou grau de satisfação (Letras: Muito Bom;Bom;Ruim;Muito Ruim). Para pesquisar digite os dados separados por Ponto e virgula ";"

OU

Importação de arquivo: Importe um arquivo do tipo (CSV) que está em seu dispositivo clicando em “Escolher arquivo”. Após a importação, deve-se pressionar o botão “Converter” para adicionar os valores na caixa de texto. (Isso permite uma maior personalização para você, pois possibilita a alteração dos dados inseridos). Escolha a variável desejada na caixa “Selecione...” Depois de ter feito os passos acima, basta premir “Calcular”.

Tipo de pesquisa

Variável nominal: Usada para descrever classes ou categorias, mas sem ordenação entre elas. Exemplos: sexo, cor dos olhos, tom de pele, doente/sadio.

Variável ordinal: Usada para descrever classes ou categorias ordenadas por você. Exemplos: escolaridade (1°, 2°, 3° graus), estágio da doença (inicial, intermediário, terminal), mês de observação (janeiro, fevereiro..., dezembro).

Variável discreta: Usada geralmente para contagens, por isso faz sentido apenas o uso de números inteiros como: número de filhos, quantidade de automóveis, número de cômodos de uma casa, etc.

Variável contínua: Usada para definir intervalos entre dois valores, por isso faz sentido o uso de números fracionais. Exemplos: peso, altura, dinheiro, tempo, idade.

Tipo de escala

População: Esta escala é utilizada com valor em escala total.

Amostra: Esta escala é utilizada para valores menores ou pequenos grupos.

Após apertar o botão calcular

Caso tenha selecionado ordinal deverá ser colocados os valores na ordem de sua escolha.

Após inserir os valores, é necessário pressionar calcular novamente.

Medidas Separatrizes

As medidas separatrizes são valores que dividem a sequência de dados.

Quartil: 1/4

Quintil: 1/5

Decil: 1/10

Porcentil:

1/100

Leitura da tabela

FI: Frequência ou repetições da variável.

FR%: Porcentagem de repetições.

FAC: Frequência acumulada.

FAC%: Porcentagem da frequência acumulada.

Moda: É a variável que contém maior repetição.

Média: Valor da soma de uma variável vezes a sua frequência dividida pela frequência acumulada, Média = variável1 x Frequência1 .../FAC.

(Nominal e Ordinal não inclui média, desvio padrão e coeficiente de variação).

Mediana: A mediana é o valor da variável que ocupa a posição central de um conjunto de n dados ordenados.

Tipos de Gráfico: É gerado gráficos de acordo com cada tipo de pesquisa.

Nominal: Tipo pizza.

Ordinal: Tipo pizza.

Discreta: Tipo barra.

Contínua: Tipo histograma.

Probabilidade Estatística

Uniforme: Utilizamos ela quando queremos encontrar a probabilidade de um evento ocorrer dentro de um intervalo.

Binomial: Utilizamos em casos de experimentos repetidos, onde existem dois possíveis resultados. (Ex.: sucesso ou fracasso).

Normal: A distribuição normal pode ser usada para aproximar distribuições discretas de probabilidade, como por exemplo a distribuição binomial. Além disso, a distribuição normal serve também como base para a inferência estatística clássica.

Como usar

Probabilidade Uniforme Estatística

Lembre-se: Números decimais devem ser representados com ponto "." (Ex: 5.23). Não devem ser representados números decimais com vírgula "," (Ex: 5,23)

Ponto mínimo: É o valor inicial do intervalo.

Ponto máximo: É o valor final do intervalo.

Tipo de escala

Menor que: É a probabilidade do resultado ser menor que o valor digitado.

Entre: É a probabilidade do resultado ser entre os valores digitados.

Maior que: É a probabilidade do resultado ser maior que o valor digitado.

Como usar

Probabilidade Binomial Estatística

Lembre-se: Números decimais devem ser representados com ponto "." (Ex: 5.23). Não devem ser representados números decimais com vírgula "," (Ex: 5,23)

N: É a quantidade da amostra pesquisada.

P: É a taxa de obter sucesso. (Use porcentagem mas sem o símbolo "%")

Q: É a taxa de obter fracasso. (Use porcentagem mas sem o símbolo "%")

Eventos: Valor do evento a ser pesquisado.

Como usar

Probabilidade Normal Estatística

Lembre-se: Números decimais devem ser representados com ponto "." (Ex: 5.23). Não devem ser representados números decimais com vírgula "," (Ex: 5,23)

Média: É o valor esperado de uma ocorrência.

Desvio padrão: Indica o quanto um conjunto de dados é uniforme. Quanto mais próximo de 0 for o desvio padrão, mais homogêneo são os dados.

Tipo de escala

Menor que: Intervalo menor do que o valor inserido a ser pesquisado.

Entre: Intervalo entre dois valores inseridos a serem pesquisados.

Maior que: Intervalo maior do que o valor inserido a ser pesquisado.

Como usar

Correlação Estatística

Conceito: É como uma variável se comporta em um cenário onde outra está variando, visando identificar se existe alguma relação entre a variabilidade de ambas.

Regressão:

Tem como resultado uma equação matemática que descreve o relacionamento entre as duas variáveis (X, Y).

X: Abscissa da regressão.

Y: Ordenada da regressão.

(Isso permite uma maior personalização para você, pois possibilita a alteração dos dados inseridos).

Depois de ter feito os passos acima, basta premir “Calcular”.

Após calcular: Caso queira fazer uma projeção futura, selecione Y e digite o valor da ordenada para obter o valor da abscissa. Ou selecione X e digite o valor da abscissa para obter o valor da ordenada.
